สรุปความรู้ที่เรียน

ปัจจุบัน คอมพิวเตอร์มีความสำคัญและมีประโยชน์ต่อมนุษย์ทั้งทางตรงและทางอ้อม
ทั้งในการใช้ชีวิตประจำวันและในเชิงธุรกิจ โดยเฉพาะในการประกอบอาชีพ
แทบทุกอาชีพมีเทคโนโลยีเข้ามาเกี่ยวข้อง เครือข่ายคอมพิวเตอร์จึงมีความสำคัญอย่างยิ่ง

แนวทางหน้าที่ในการประกอบอาชีพในอนาคต

ภายหลังเรียนจบ สามารถประกอบอาชีพที่เกี่ยวข้องกับคอมพิวเตอร์ได้หลายรูปแบบ
เช่น อาชีพที่ใช้ความสามารถในการเขียนโปรแกรม Coding พัฒนาระบบ software ต่างๆ เป็นต้น

อาชีพตัวอย่างในสายคอมพิวเตอร์

Programmer หรือ นักเขียนโปรแกรม ทำหน้าที่เขียนโปรแกรมคอมพิวเตอร์
โดยใช้ภาษาทางคอมพิวเตอร์ เช่น HTML Java Python

Developer นักพัฒนาโปรแกรมและเว็บไซต์ เป็นสายงานที่ต่อยอดมาจากFโปรแกรมเมอร์

Software Engineer มีหน้าที่ทำงานวิจัย ออกแบบ พัฒนา เขียน Coding

Data Scientist มีหน้าที่ในการนำข้อมูลมาวิเคราะห์ด้วยเทคนิคขั้นสูง
